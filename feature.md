# Feature

1. feature ask
fitur ini untuk memberi dan menyimpan jawaban dari user, dimana akan diberikan pertanyaan kepada user, kemudian menerima pesan dari user.

2. feature progress
feature ini menyimpan progress user, user akan mengirimkan progress dan akan disimpan oleh sistem.

3. feature penyemangat
fitur ini akan memberi list penyemangat untuk user, berdasarkan  jawaban user dan progress dari user untuk memberi respon ke user.
